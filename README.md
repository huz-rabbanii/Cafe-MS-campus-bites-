# CampusBites

CampusBites is a platform designed to streamline food ordering and delivery services within a campus environment. It connects students, campus eateries, and delivery personnel for a seamless experience.

## Features

- **User-Friendly Interface**: Easy navigation for students, eateries, and delivery personnel.
- **Order Management**: Real-time order tracking and updates.
- **Payment Integration**: Secure and multiple payment options.
- **Notifications**: Instant updates on order status.
- **Admin Dashboard**: Manage users, orders, and eateries.

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd CampusBites
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## Technologies Used

- **Frontend**: React, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Authentication**: JWT
